# DisnApp

Red social tipo feed desarrollada con **ASP.NET Core MVC**, **Entity Framework Core** e **Identity**.

## Funcionalidades
- Registro / Login (Identity)
- Crear publicaciones (Descripción + URL de imagen)
- Feed ordenado por fecha
- Edit / Delete (en progreso)
- Likes y comentarios (en progreso)

## Tecnologías
- ASP.NET Core MVC
- Entity Framework Core
- ASP.NET Identity
- Bootstrap 5
- SQL Server
